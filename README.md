# Clinic Appointment Scheduler

A simple Clinic Appointment Scheduler developed in C++ using Linked Lists and Queue data structures. This project helps manage patient appointments, maintain records, and organize the appointment queue in an easy and efficient way.

## Features

- Add new patients
- Delete patient records
- Search patients by ID
- Search patients by name
- Update patient information
- Display all patient records
- Display pending patients
- View appointment queue
- Serve the next patient
- Sort records by appointment date
- Generate clinic reports

## Data Structures Used

- Linked List
- Queue

## Programming Language

- C++

## Concepts Used

- Classes and Objects
- Structures
- Dynamic Memory Allocation
- Linked List Operations
- Queue Operations
- Searching
- Sorting
- Functions
- File Organization
